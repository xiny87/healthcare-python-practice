# Project: Hospital Readmission Exploratory Data Analysis

This project performs exploratory data analysis (EDA) on a simulated hospital dataset.

The notebook demonstrates how Python and pandas can be used to inspect, summarize, and analyze healthcare-related data.

---

# Project Objective

Hospital readmission is an important indicator of healthcare quality and patient outcomes.  
This exercise explores a simplified hospital dataset to practice identifying patterns that may be associated with patient readmission.

The goal of this analysis is to practice core data analysis workflows, including data inspection, summarization, grouping, filtering, and basic exploratory analysis.

---

# Dataset

The dataset simulates hospital patient visit information and contains the following variables:

- **patient_id** – unique identifier for each patient  
- **age** – patient age  
- **gender** – patient gender  
- **visit_date** – date of hospital visit  
- **readmitted** – whether the patient was readmitted (1 = yes, 0 = no)

---

# Key Analysis Steps

The notebook demonstrates several fundamental exploratory data analysis techniques:

- Inspecting the dataset structure
- Checking for missing values
- Generating summary statistics
- Sorting and filtering patient records
- Grouping and aggregating data
- Calculating readmission rates
- Creating contingency tables for categorical comparisons

These steps represent common workflows used in healthcare data analysis.

---

# Example Analysis Questions

This exploratory analysis investigates questions such as:

- What is the distribution of patient age?
- How many patients were readmitted?
- Are there differences in patient characteristics across gender groups?
- What patterns appear when comparing gender and readmission status?
